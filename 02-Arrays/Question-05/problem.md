# Left Rotate an Array by One

## Given an integer array nums, rotate the array to the left by one.

### Rotating the array left by one means shifting all elements by one place to the left and moving the first element to the last position of the array.

### Example 1:

```text
Input: nums = [1,2,3,4,5]
Output: [2,3,4,5,1]
```
